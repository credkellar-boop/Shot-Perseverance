# ⚾ Shot-Perseverance OS

![Build Status](https://img.shields.io/github/actions/workflow/status/darionkellar/shot-perseverance/python-app.yml?branch=main&style=for-the-badge&logo=github)

**Shot-Perseverance OS** is a 2026-gen computer vision engine for multi-sport analytics.

### 🏟 Supported Sports
* **⚾ Baseball:** Pitch velocity, break-point calculation, and bat-speed dynamics.
* **🏀 Basketball:** Release height, "The Dip" rhythm, and Alley-oop detection.
* **⚽ Soccer:** Offside line visualization and "Beckham Curve" trajectory prediction.

### 🛠 Tech Stack
* **Vision:** OpenCV & MediaPipe
* **Physics:** NumPy (Magnus Effect calculation)
* **Reporting:** FPDF2 & Matplotlib
