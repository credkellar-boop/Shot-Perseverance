### 🏟 Multi-Sport Kernel
The engine dynamically adjusts physics constants based on the active sport:

* **⚾ Baseball:** Pitch velocity tracking, break-point calculation, and swing power analytics.
* **🏀 Basketball:** "The Dip" rhythm analysis, release angle, and Alley-oop detection.
* **⚽ Soccer:** Real-time offside line visualization and ball-curve prediction.
* **🥊 Combat:** Strike impact frame analysis and contact dynamics.

---

### 🚀 Quick Start
1. **Install:** `pip install -r requirements.txt`
2. **Calibrate:** `python core/calibration.py`
3. **Run:** `python main.py`
